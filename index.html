Periodic Table Tower Defense

This is pttd, a tower defense game based on the periodic table. Press Launch to launch the game.

The following buttons might be helpful:

  • u: select first in-game button (first upgrade)
  • p or escape: pause the game
  • space: skip to the next wave
  • f1: you are a cheater

You must have Java enabled to play PTTD.
